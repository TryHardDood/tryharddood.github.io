Get Started

AdvancedKitsReloaded is a Kit managing plugin for Minecraft servers which runs Spigot or CraftBukkit 1.8, 1.9 and 1.10.
With this plugin you can easily create/edit/delete your kits and you can customize it as much as you'd like.

Features
  1. Timed cooldown - You can restrict players from using a kit continuously. (Ex: 0.5 = half an hour )
  2. GUI for kits - You can view and buy/use kits from a simple GUI.
  3. Economy support - You can use your economy plugin, but it have to support Vault
  4. CustomEnchantments support - Works most of the time
  5. Custom permissions - You can modify the default permissions per kit
  6. Run commands after using a kit - After the player got his/her kit a command will run
  7. Custom kit icon - You can change the icon of the kit in the GUI
  8. Update checks If an update comes out, you'll get a notification
Dependencies

If you would like to use the buying feature you'll have to have Vault installed.

Installation
  1. Download AdvancedKitsReloaded
  2. Make sure your server is not running.
  3. Copy the .jar file into your plugins directory.
  4. Start the server.
  5. If you made a backup of your config.yml file, stop the server and edit the newly generated config.yml file with only what you need, from the backup.
  6. Start the server.
  7. Enjoy!
Commands
Command Permission Details
/kit advancedkits.kit.use Gives access to the gui.
/kit use <kitname> advancedkits.kit.use Gives access to use kits
/kit buy <kitname> advancedkits.kit.buy Gives access to buy kits
/kit view <kitname> advancedkits.kit.view Gives access to view kits
/kit give <player> <kitname> advancedkits.give Gives access to give kits
/kit edit <kitname> advancedkits.kitadmin Gives access to edit kits
/kit create <kitname> advancedkits.kitadmin Gives access to create kits
/kit update advancedkits.kitadmin Downloads and updates to the latest version.
/kit delete <kitname> advancedkits.kitadmin Gives access to create kits
/kit reload advancedkits.reload Gives access to reload the configuration
/kit setflag <kit> <flag> <value> advancedkits.kitadmin Gives access to manage flags of the kits
/kit edititem <variable> <value> advancedkits.kitadmin Gives access to edit items in your hand
Configuration

The configuration of this plugin is very simple:

# +----------------------------------------------------+
# <               AdvancedKits Reloaded                >
# <                 Configuration file                 >
# <                     config.yml                     >
# +----------------------------------------------------+
# You can enable the economy hook via Vault
# You need an Economy plugin, like Essentials
use-economy: true
# If this option is true, the player who buys a kit will automatically use it.
use-on-buy: false
# You can change the chat prefix.
# So it's looks more unique :)
chat-prefix: '&7[&6AdvancedKits&7]'
# AdvancedKitsReloaded support multilanguage.
# Currently there are the available values:
# English: en | Hungarian: hu | Portoguese: pt | Italian: it | Finnish: fi | French: fr
# If you'd like to translate it here:
# https://ackuna.com/translate-/advancedkits-reloaded/
locale: en

use-economy - If you would like to enable the buying feature you have to have [Vault][1] and this option enabled.
chat-prefix - This prefix will show in chat
locale - This plugin is supports more languages
use-on-buy - This option allows you to enable automatic kit use on buy

Flags and Variables
Kit Flag
Flag Default value Details
visible true Displays the kit in the GUI.
permonly false Players with permission can use this kit.
firstjoin false If this flag is enabled the newbies recieve this kit automatically after joining.
addworld, delworld [] You can’t use kits in these worlds.
icon EMERALD_BLOCK Displays the kit with the icon in the GUI.
unlocked false Auto unlocks the kit.
displayname Kit’s name Displays the kit with the given name (colors supported).
permission advancedkits.kit.use.[kitname] Players with permission can use this kit. (if permonly enabled)
cost 0 The cost of the kit.
delay 0.0 The delay for the kit.
addcommand, delcommand [] The list of commands which will run when a player used this kit.
uses 0 The maximum uses of this kit per player.
clearinv false Whether or not clear the player inventory when using this kit.
Item variables
Flag Default value Details
name   You can set the item’s displayname (Colors supported)
addlore, dellore   You can add/remove lore for an item
amount   You can set the item’s amount in your hand
durability   You can set the item’s durability
