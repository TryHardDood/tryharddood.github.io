Get Started

Made by TechGuard and updated by TryHardDood. This plugin allow you to setup custom zones for example shops, pvp areas or spawn. The zones can be created by a player. ( If he/she has a specified permission. )

Features
  1. Zone protection
  2. Easy-to-use
  3. Update notifier
  4. 1.7.10, 1.8.X, 1.9.4 and 1.10 support. ( It protects your ArmorStands and ItemFrames also. )
  5. Multilanguage support ( Send me translations! Currently only “en” and “hu” available. )
  6. Players can use it
  7. Useful flags ( like ‘fly’ or ‘water-flow’ )
Installation
  1. Download iZone
  2. Make sure your server is not running.
  3. Copy the .jar file into your plugins directory.
  4. Start the server.
  5. If you made a backup of your config.yml file, stop the server and edit the newly generated config.yml file with only what you need, from the backup.
  6. Start the server.
  7. Enjoy!
